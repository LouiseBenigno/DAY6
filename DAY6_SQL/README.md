Learning Journal Entry #6

 The Messy Dataset: I was surprised by how chaotic real data is; I didn't expect so many typos, random spaces, and test rows hidden in a professional database.
 Tool Preference: I prefer SQ because writing queries gives me precise control over changes, though Power Query is faster for visual filtering.
  Hardest Issue: Standardizing the inconsistent text (like typos in cities and categories) was the toughest part because it required carefully hunting down every variation first.
Importance of Documentation: Documenting steps is crucial because it creates an "audit trail"—I would tell a manager it proves our final reports are accurate and can be replicated by anyone.
 Real Job Impact: These skills are essential for a real job because clean data is the only way a company can make accurate, trustworthy business decisions.
